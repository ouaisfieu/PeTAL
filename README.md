# PTAL OS

PTAL OS est un système documentaire citoyen statique.

Il permet de :
- produire
- lire
- contextualiser
- diffuser

des documents complexes, sourcés et critiques,
sans serveur, sans dépendance lourde.

## Principe
Le Shell ne contient aucun contenu.
Tout est chargé via registry.json.
